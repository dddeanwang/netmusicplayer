# 仿云音乐播放器

## 后台接口:网易云音乐NodeJS版API
https://binaryify.github.io/NeteaseCloudMusicApi/#/

## 功能
- [x] 发现页
- [x] 推荐歌单
- [x] 最新音乐
- [x] 最新MV
- [x] 搜索热搜榜
- [x] 搜索历史
- [x] 搜索详情
- [x] MV详情
- [x] 专辑详情
- [x] 歌手详情
- [x] 播放列表
- [x] 播放记录
- [x] 歌单详情
- [x] 查看评论
- [x] 歌词滚动

## 技术栈
- Vue2全家桶
- ElementUI(v-loading,pagination,el-tabs...)
- Lazyload
- Axios
- Sass

## 安装与使用

npm install
npm run serve
